!! Atenção - Essa automação limpa a pasta de Downloads, então retire Arquivos necessários antes de inicia-la. !!

Para que a automação funcione, é preciso seguir esses passos:

1. Instalar o Python na Maquina - ( https://www.python.org/ftp/python/3.13.1/python-3.13.1-amd64.exe )

2. Executar o "Instalador" - instalação de configurações necessárias e clone do repositório.

3. Executar o "Credenciais de Acesso" - Coleta as credenciais para Login do Portal e Servidor IMAP.

4. Inserir Usuário do Portal de Boletos.

5. Inserir Senha do Portal de Boletos.

6. Inserir E-mail em que os arquivos .csv estão.

7. inserir Senha de App.
(caso não tenha a Senha de App, ative a autenticação de dois fatores e acesse - https://myaccount.google.com/apppasswords )

9. Deve selecionar a pasta de Downloads da maquina e salvar.

10. Após ter feito esses procedimentos apenas inicialize o "Macro Portal de Boletos".

- Em caso de atualização dos Arquivos utilize "Atualizar" para que os arquivos sejam atualizados.
